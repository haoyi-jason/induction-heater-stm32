# Induction Heater STM32

## Project Overview
This project is designed to develop a highly efficient induction heater using STM32 microcontroller. The aim is to create a soldering device that can heat up metal components quickly and uniformly.

## Features
- Adjustable temperature control
- Safety features to prevent overheating
- User-friendly interface using display and controls
- Real-time temperature monitoring

## Components Used
- STM32 Microcontroller
- Induction coil
- Temperature sensors (e.g., thermocouple)
- Power supply unit
- Control buttons

## Installation Instructions
1. Clone the repository:  
   `git clone https://github.com/haoyi-jason/induction-heater-stm32`

2. Open the project in your preferred IDE.
3. Connect the STM32 development board.
4. Upload the code to the board.

## Usage
- Power on the device and set the desired temperature using control buttons.
- Monitor the temperature on the display.
- Use safety precautions while handling heating components.

## Troubleshooting
- If the device does not power on, check the power supply connection.
- Ensure all components are properly connected.
