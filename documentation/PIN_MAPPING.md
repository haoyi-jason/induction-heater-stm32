# PIN_MAPPING

This document describes the pin assignments and connections for the induction heater project.

## Pin Assignments

| Pin Number | Function     | Description                   |
|------------|--------------|-------------------------------|
| 1          | VCC          | Power supply (+3.3V)         |
| 2          | GND          | Ground connection             |
| 3          | PWM          | Control signal for induction  |
| 4          | TEMP_SENSOR  | Temperature sensor connection  |
| 5          | LED          | Status indicator LED          |
| 6          | BUZZER       | Buzzer for alerts             |

## Connections
- Connect **Pin 1** to the power supply.
- Connect **Pin 2** to the ground.
- Connect **Pin 3** to the PWM controller.
- Connect **Pin 4** to the temperature sensor module.
- Connect **Pin 5** to the LED indicator.
- Connect **Pin 6** to the buzzer for alerts.

Make sure to verify all connections before powering up the system.
